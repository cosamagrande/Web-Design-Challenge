Summary: Latitude vs. X


Purpose of project is to analyze changes in weather as we approach the equator. To accomplish this, data was pulled from the OpenWeatherMap API to gather a dataset from more than 500 cities.
Once the data set is gathered, pandas Matplotlib is used to plot various weather metrics versus latitude. Included metrics are Temperature, cloudiness, wind speed and humidity. This site presents visualizations of data trends along with the sourced data, along with explanations observed from trends.
